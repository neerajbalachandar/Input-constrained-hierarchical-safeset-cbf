# Input-constrained-hierarchical-safeset-cbf

A fixed barrier ℎ(𝑥) can still work under constrained inputs as long as the required safe action lies inside the admissible input set.

iccbf mat - contains original paper codes modified to generate more plots on input constrained cbf.

iccbf py - just a failed attempt to convert to py

cbf_obs, inner_safe_set - codes written by us for normal cbf and to visualize inner safe sets, without any sos
